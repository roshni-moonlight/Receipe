# Spice Hub Recipe Website

## Project Description

Spice Hub is a recipe website developed using PHP, HTML, CSS, and MySQL. The website provides various food recipes including breakfast, biryani, grilled dishes, seafood, sandwiches, pizzas, burgers, shawarma, and more.

## Features

* View different food categories
* Display recipe details
* CRUD operations using PHP and MySQL
* Responsive user interface
* Food images and recipe descriptions

## Technologies Used

* HTML
* CSS
* PHP
* MySQL
* XAMPP
